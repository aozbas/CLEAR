# Skin Lesion Identification App

Educational app for classifying skin lesions via ML.

## Stack
- **mobile/** — React Native + Expo + TypeScript
- **backend/** — Python FastAPI (auth, uploads, predictions, history)
- **ml/** — PyTorch training & inference
- **Infra** — Supabase (auth, Postgres, storage)

## Phases
1. Planning
2. Backend foundation
3. ML integration (supervised first; RL deferred)
4. Mobile app
5. Improvements

## Principles
Keep it simple. Clear separation between mobile / backend / ml. Understandable top-to-bottom.
